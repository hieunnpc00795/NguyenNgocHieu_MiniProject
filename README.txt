		MINI PROJECT CART

	Chào công ty R2S em là Nguyễn Ngọc Hiếu hiện là sinh viên ngành ứng dụng phần mềm trường cao đăng FPT Polytechnic Cần Thơ.

	Giới thiệu: 
		- Đây là mini project của em được làm theo yêu cầu của công ty với chủ đề tự chọn theo 12 yêu cầu từ file Excel mà công ty gửi cũng như từ buổi họp mặt đầu tiên với công ty.
		- Theo các yêu cầu mà phía công ty đưa ra trên file excel em đã chọn đề tài làm giỏ hàng cùng với module tài khoản người dùng với các chức năng:
			+ Thêm sản phẩm vào giỏ hàng.
			+ Thực hiện check out giỏ hàng.
			+ CRUD user, CRUD role, login, logout.
		- Mini project được em thực hiện trên công cụ eclipse, mysql và được xây dựng theo mô hình Sping MVC có sử dụng swagger để document các api.

	Hướng dẫn cài đặt: 
		- Sau khi dowload project từ github về máy thì tiến hành chạy file script MiniStore để khởi tạo database.
		- Tiếp tục chạy file script Data để khởi tạo các data test.
		- Mở eclipse và add project vừa tải về.
		- Mở project vừa add vào và tiếp tục mở file application.properties để thay đổi pring.datasource.username ,spring.datasource.password, spring.mail.username, spring.mail.password.
	Hướng dẫn sử dụng:
		- Sau khi tiến hành xong các bước cài đặt thì sẽ khỏi chạy project.
		- Vào brower và gõ http://localhost:8080/home/index.
		- Sử dụng các chức năng sau khi đã hiển thị giao diện.

	Cảm ơn đã xem.
